Production reliability usually fails at interfaces, not core logic.

```ts
export const idempotencyKey = (tenantId: string, eventId: string) => `${tenantId}:${eventId}`;
```

Three patterns consistently help:

1. Admission control before expensive work.
2. Idempotent writes with bounded memory windows.
3. Latency-focused dashboards with tail metrics, not averages.
