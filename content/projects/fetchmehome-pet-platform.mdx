## Problem

Pet adoption flows and lost-pet recovery are fragmented across social media groups and informal chat channels, making discovery and trust difficult.

## Constraints

- Need for simple UX for non-technical users
- Verified listing flows to reduce spam/scams
- Search relevance across breed, age, and location

## Approach

Built a unified platform for adopters, rescuers, and owners to post adoption listings and missing-pet alerts with profile-based credibility.

## Architecture

- React frontend with protected routes and profile dashboards
- Express/Node API for listings, messaging metadata, and auth
- MongoDB schema tuned for geo and status filters

## Results

- Location-aware matching with filtered search across adoption and recovery workflows.
- Increased listing discoverability through multi-filter search and status tags.

## What I'd Do Next

Add real-time notifications and map-driven radius alerts for lost-pet reports.
